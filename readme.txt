This plugin injects comment code into the head section of your HTML output.

== Description ==
This plugin adds one or multiple lines of comment code in the HEAD section of your HTML output. With this plugin you can add your personal message such as your company details, job invitation or a funny gimmick into your code.

= Features =

* Add comment code to your head section
* Live preview of the output
* Plug and play

= Links =
* [Plugin's web page](http://www.prodes.nl/wordpress/builder-source/)

== Installation ==
This section describes how to install and use the plugin.

1. Install automatically through the `Plugins` menu and `Add New` button (or upload the entire `builder-source` folder to the `/wp-content/plugins/` directory)
2. Activate the plugin
3. Go to `Settings` => `Builder Source`
4. Add your personal message
5. Hit the save button
6. Check the source of your website :)
